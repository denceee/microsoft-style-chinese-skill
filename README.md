# microsoft-style-chinese

> 坐和放宽，好东西就要来了！

一个 [WorkBuddy](https://www.codebuddy.cn/) AI 技能（Skill），让 AI 以**「微软式中文」**风格进行表达。

## 什么是微软式中文？

微软式中文是指微软公司在软件产品中文本地化过程中，因大量使用机翻及人工翻译质量不佳而产生的**怪异别扭、语法混乱、拗口难懂但语义尚可揣摩**的中文表达风格。该风格已成为中文互联网经典迷因。

经典名场面：

| 微软式中文 | 正常中文 | 英文原文 |
| --- | --- | --- |
| 坐和放宽 | 请稍候 | Sit back and relax |
| 头抬起 | 注意 | Heads up |
| 滚回以前的版本 | 回退到以前的版本 | Roll back |
| 好东西就要来了！ | 新功能即将推出 | Good things are coming |
| 你正在成功 | 正在处理 | You're succeeding |
| 请勿™关闭计算机 | 请不要关闭计算机 | Please do not turn off your computer |
| 功率 | 电源 | Power |
| 微软边缘 | Edge 浏览器 | Microsoft Edge |
| 凉！ | 酷！ | Cool! |
| 免费花分文 | 完全免费 | Free / Zero / Nada |

更多内容见 [萌娘百科 - 微软式中文](https://mzh.moegirl.org.cn/微软式中文)

## 功能特性

- 🎭 **风格转换**：将 AI 回复转换为微软式中文风格，保留内容准确性
- 📖 **完整用语对照表**：30+ 经典微软式中文翻译收录
- 📝 **句式模板**：开场白、操作提示、报错、结束语等模板
- 🎚️ **风格浓度控制**：支持轻度、中度、重度三种浓度
- 🔧 **造词规则**：没有现成梗时可按规则自动生成微软式表达

## 安装方式

### 方式一：手动安装

将本仓库的文件复制到 WorkBuddy 技能目录：

```
~/.workbuddy/skills/microsoft-style-chinese/
├── SKILL.md
└── references/
    └── style_guide.md
```

### 方式二：打包安装

下载 `microsoft-style-chinese.zip`，在 WorkBuddy 中通过技能管理界面导入。

## 使用方法

安装后，在 WorkBuddy 对话中以下任一方式触发：

- "用微软式中文回答…"
- "微软式中文风格…"
- "像微软翻译一样说话"
- "坐和放宽风格"
- 直接引用"坐和放宽"、"头抬起"、"好东西就要来了"等经典梗

## 示例

**用户：** 帮我看看怎么把电脑恢复出厂设置

**AI（微软式中文）：**
> 头抬起！为了确保你是最新的，我们将要对你的计算机进行重置。请坐和放宽，你正在成功。首先，请打开你的功率，然后进入设置，找到"恢复"选项，点击"重置此计算机"。在此过程中，请勿™关闭计算机。如果你遇到不顺利，没关系，我们都有不顺利的时候。你可以滚回到以前的版本，或者打开微软边缘浏览器进入微软官网进行反映。好东西就要来了！

## 技术细节

- **技能类型**：用户级 Skill（跨项目可用）
- **触发机制**：基于 `description` 元数据的语义匹配
- **依赖**：无额外脚本依赖，纯文本规则驱动
- **兼容平台**：WorkBuddy 所有支持 Skill 的平台
