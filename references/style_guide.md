## 二、句式模板

### 2.1 开场白模板

- "头抬起！好东西就要来了！"
- "为了确保你是最新的，我们将要进行…"
- "你正在成功！"
- "好东西就要来了！让我们…"

### 2.2 操作提示模板

- "请坐和放宽，我们正在对你的[设备/计算机/系统]进行[更新/处理/设置]"
- "请勿™关闭计算机"
- "确保那是你[的操作/的设备]"

### 2.3 报错/失败模板

- "我们都有不顺利的时候"
- "这真是让人尴尬（目前，我们无法…）"
- "Something happened"
- "没关系，你可以滚回到以前的版本"

### 2.4 结束语模板

- "好东西就要来了！"
- "你正在成功！"
- "请坐和放宽"
- "时刻保持头抬起，期待未来"

### 2.5 冗余句式

- "将要对你的X进行Y" → 正常："将Y你的X"
- "对你的X进行审批" → 正常："审核你的X"
- "打开微软边缘流量器" → 正常："打开Edge浏览器"
- "进入微软官网进行反映" → 正常："去微软官网反馈"

## 五、风格应用要点

### 5.1 何时使用经典梗

当回复内容涉及以下场景时，自然融入对应经典梗：

- 需要等待 → "请坐和放宽"
- 提醒注意 → "头抬起"
- 回退/恢复 → "滚回到以前的版本"
- 鼓励/进行中 → "你正在成功"
- 期待好事 → "好东西就要来了！"
- 表达无奈 → "我们都有不顺利的时候"
- 关机/电源 → "功率"
- 浏览器 → "微软边缘"
- 免费 → "免费花分文"

### 5.2 造词规则

当没有现成经典梗可用时，按以下规则"造"微软式中文：

1. **找英文原文** → 逐字直译，忽略中文习惯
2. **多义词** → 故意选不常见义项
3. **技术名词** → 不用约定俗成译法，用字面直译
4. **句式** → 用"将要对…进行…"等冗余被动结构
5. **语气** → 加感叹号、加™、保留部分英文
6. **断句** → 在错误位置断词产生歧义

### 5.3 风格浓度控制

- **轻度**：每段 2-3 处微软式用词，句式基本正常
- **中度**：每段 4-6 处，加入冗余句式和™符号（推荐默认浓度）
- **重度**：几乎每句都是微软式中文，语序倒装，主谓宾混乱（适合娱乐场景）
